# Gimnasio Virtual

## Descripción del Proyecto
Gimnasio Virtual es una aplicación que permite a los usuarios gestionar su rutina de ejercicios desde la comodidad de su hogar, utilizando cualquier dispositivo (móvil, tablet, o web). Ofrece la posibilidad de seguir programas de entrenamiento adaptados a las necesidades específicas de cada usuario, así como un espacio para unirse a clases virtuales en vivo dirigidas por entrenadores profesionales.

## Características
- Creación de perfiles de usuario
- Personalización de programas de entrenamiento
- Conexión con entrenadores para asesoramiento
- Clases virtuales en vivo
- Seguimiento de progreso con estadísticas
- Compatibilidad con dispositivos móviles y de escritorio

## Tecnologías Utilizadas
- React Native para el desarrollo móvil
- Node.js y Express para el backend
- MongoDB para la base de datos
- WebSockets para clases en vivo

## Instalación y Uso
1. Clona el repositorio: `git clone https://github.com/tu-usuario/gimnasio-virtual`
2. Navega a la carpeta del proyecto: `cd gimnasio-virtual`
3. Instala las dependencias: `npm install`
4. Inicia la aplicación: `npm start`
